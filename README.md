  <b>AcherLab Panel</b> — AI-powered VPS management dashboard.<br/>
  Monitor servers, manage tunnels, chat with AI assistant. All in one place.
</p>

---

## Quick Start

```bash
git clone https://github.com/acherlab-ai/AcherLab-Panel.git
cd AcherLab-Panel
docker compose up -d
```

Open **http://localhost:8000** in your browser.

## Features

- **AI Assistant** — Built-in AI chatbot for server management
- **System Monitor** — Real-time CPU, Memory, Disk stats
- **Tunnel Manager** — View and manage TCP tunnels
- **Service Status** — Live monitoring of all AcherLab services
- **Quick Actions** — One-click tunnel creation, RDP access

## API Endpoints

| Endpoint | Description |
|----------|-------------|
| `GET /` | Service info |
| `GET /api/system` | System stats (CPU, RAM, Disk) |
| `GET /api/services` | Service status list |
| `POST /api/ai/chat` | AI chat assistant |
| `GET /api/tunnels` | Tunnel list |
| `GET /api/health` | Health check |

## Deploy

```bash
git clone https://github.com/acherlab-ai/AcherLab-Panel.git
cd AcherLab-Panel
docker compose up -d
```

Open http://localhost:8000

## Tech Stack

`FastAPI` `Tailwind CSS` `Docker` `Python 3.11` `psutil`
